HTML o'zi nima? HTMLning kelib chiqish tarixiga nazar.

HyperText Markup Language (HTML) — veb-brauzorda ko'rsatiladigan hujjatlar (veb-sahifalar) tuzilishini belgilash tili. U CSS (Cascading Style Sheets) bilan birga ko'rinishni bezashda, JavaScript esa mantiqiy ishlarni bajarishda yordam beradi. Masalan, HTML — suyak, CSS — teri/ko'rinish, JavaScript — mushak/harakat vazifasiga o'xshatiladi.

HTML tarixi

HTML 1989–1992-yillarda Yevropa Yadro Tadqiqotlar Markazida (CERN) ishlagan ingliz olimi Tim Berners-Li tomonidan taklif qilingan va birinchi marta qo'llanilgan. Dastlab u ilmiy ma'lumotlarni oson ulashish va bog'lash (hipermatn) uchun ishlab chiqilgan edi. Keyinchalik butun dunyo bo'ylab veb-texnologiyalarning asosiga aylandi.

Sintaksis

HTML hujjati teglardan iborat. Quyida juda sodda skelet keltirilgan:

<!DOCTYPE html>
<html>
  <head>
    <title>Sarlavha</title>
  </head>
  <body>
    <h1>Salom HTML!</h1>
    <p>Bu yerda matn bo'ladi.</p>
  </body>
</html>

Imkoniyatlar

HTML quyidagi formatlash imkoniyatlariga ega:

  • Matn elementlari: h1..h6 sarlavhalar, p paragraf, br yangi qatordan yozish.
  • Rasm, audio, video qo'shish: img, audio, video.
  • Gavdalantirish va bog'lanish: a (giperhavola), nav, header, footer.
  • Ro'yxatlar: ul / ol va li elementlari.
  • Jadvallar: table, tr, th, td.
  • Formalar: form, input, textarea, button.
